# Innovation and Clean Transport

## Overview
This project analyzes the relationship between economic development and carbon emissions across countries.

## Data
- GDP and CO2 emissions dataset (Kaggle)

## Method
- Filtered dataset for the year 2020
- Created scatter plot comparing GDP per capita and CO2 emissions per capita

## Key Finding
Countries with higher GDP per capita tend to have higher CO2 emissions per capita, indicating that economic growth is often associated with increased environmental impact.

## Policy Insight
This suggests the importance of investing in clean technologies and innovation to decouple economic growth from emissions.
